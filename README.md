Для запуска проекта "Моделирование движения тел в двухмерной среде (с учетом гравитации)" необходима среда разработки "Visual Studio 2022" и библиотека SFML 3.0.0, 
которая заархивирована внутри проекта - "External.7z". Для запуска необходимо разархивировать "External.7z" в проект и удалить сам архив из тела проекта.
Далее с помощью среды разработки открыть файл "CuRsOvAyA.sln", далее, выбрав язык программирования C++ 17, убедившись что проект правильно расположен в рабочей директории
запускаем файл main.cpp.

Горячие клавиши внутри программы:
Клавиша	Действие
A	      Добавить тело
C	      Очистить сцену
Space	  Пауза / Продолжить
M	      Настройка массы
F	      Настройка трения
S	      Выбор формы

Элементы UI:
Элемент	      Описание	                    Управление
Кнопки	      Добавление, очистка, пауза	  Мышь / A, C, Space
Слайдеры	    Настройка массы, трения      	Перетаскивание мышью
Индикаторы	  Статус симуляции            	—
